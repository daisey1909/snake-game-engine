# snake-game-engine
A Snake Game built using Python, Pygame, and OOP concepts
# 🐍 Snake Game Engine

## 🚀 Technologies Used
- Python
- Pygame
- Object-Oriented Programming (OOP)

## 🎮 Features
- Multiple difficulty levels (Easy, Medium, Hard)
- Bonus food system (extra points)
- Score tracking with high score saving
- Pause and resume functionality
- Smooth gameplay and animations

## 🎯 Controls
- Arrow Keys / WASD → Move snake
- P or ESC → Pause game
- ENTER → Restart game

## ▶️ How to Run
1. Install pygame:
   pip install pygame

2. Run the game:
   python snake_game.py
